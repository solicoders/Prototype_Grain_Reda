# Jeux test

## Création des Models

Model pour Models

- `php artisan make:model Models`

Model pour Cars

- `php artisan make:model Cars`

## Création de Seeders

Seeder pour Models

- `php artisan make:seeder ModelsSeeder`

Seeder pour Cars

- `php artisan make:seeder CarsSeeder`

## Lancer des Seeders

- `php artisan db:seed`
